Product Vision

CyValidator is designed as a startup-style cyber validation platform. Development is done on Windows, while final deployment will be prepared for Ubuntu Server. Future modules will validate security controls, detect configuration drift, calculate enterprise risk, simulate attack paths and track remediation from detection to closure.
